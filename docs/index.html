PennSim Simulator

A continuation on the original educational microcomputer.

This project was created to continue on and in the future improve the features and usability of the PennSim LC3/P37X Simulator. Use the side bars to navigate the project and please help out wherever you can.

If you would like to help with development, fork and review the code. Also look at any open issues. Finally, if you feel there is a feature that is missing, open a feature request in the issues and push the code for review and consideration.

Features

Improved Code

Both code and documentation have been improved to facilitate development and increase readability

Multiple Languages

Support for multiple low-level languages including LC3, P37X, as well as more to come, ensures that whatever your desired platform is you can effectively teach the skills you desire.

GitHub CI

Continuous building and testing using GitHub's own Mavin Continued Integration through the all new GitHub Actions feature.

Codacy Checking

Constant code review by Codacy makes vetting committed code faster and more efficient than ever before
